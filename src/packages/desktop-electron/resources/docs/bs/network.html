Mreža

Konfigurirajte proksije i prilagođene certifikate.

vanilla-closedcode podržava standardne proxy varijable okruženja i prilagođene certifikate za enterprise mrežna okruženja.

Proksi

vanilla-closedcode poštuje standardne varijable proxy okruženja.

# HTTPS proxy (recommended)


# HTTP proxy (if HTTPS not available)


# Bypass proxy for local server (required)
TUI komunicira sa lokalnim HTTP serverom. Morate zaobići proxy za ovu vezu kako biste spriječili petlje usmjeravanja.
Možete konfigurirati port servera i naziv hosta koristeći [CLI flags](/docs/cli#run).

Autentikacija

Ako vaš proxy zahtijeva osnovnu autentifikaciju, uključite vjerodajnice u URL.
Izbjegavajte tvrdo kodiranje lozinki. Koristite varijable okruženja ili sigurno skladište vjerodajnica.
Za proxy servere koji zahtijevaju naprednu autentifikaciju poput NTLM ili Kerberos, razmotrite LLM Gateway koji podržava vašu metodu autentifikacije.

Prilagođeni certifikati

Ako vaše preduzeće koristi prilagođene CA-ove za HTTPS veze, konfigurirajte vanilla-closedcode da im vjeruje.

Ovo radi i za proxy veze i za direktan pristup API-ju.
