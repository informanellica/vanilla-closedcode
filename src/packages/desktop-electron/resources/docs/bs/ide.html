IDE

Ekstenzija vanilla-closedcode za VS Code, Cursor i druge IDE

vanilla-closedcode se integriše sa VS kodom, Cursor-om ili bilo kojim IDE-om koji podržava terminal. Samo pokrenite closedcode u terminalu da započnete.

Korištenje

  • Brzo pokretanje: Koristite Cmd+Esc (Mac) ili Ctrl+Esc (Windows/Linux) da otvorite vanilla-closedcode u prikazu podijeljenog terminala ili fokusirajte postojeću terminalsku sesiju ako je već pokrenuta.
  • Nova sesija: Koristite Cmd+Shift+Esc (Mac) ili Ctrl+Shift+Esc (Windows/Linux) da započnete novu vanilla-closedcode terminalsku sesiju, čak i ako je ona već otvorena. Takođe možete kliknuti na dugme vanilla-closedcode u korisničkom sučelju.
  • Svijest o kontekstu: Automatski dijelite svoj trenutni odabir ili karticu s vanilla-closedcode.
  • Prečice za referencu datoteka: Koristite Cmd+Option+K (Mac) ili Alt+Ctrl+K (Linux/Windows) za umetanje referenci datoteka. Na primjer, @File#L37-42.

Instalacija

Da biste instalirali vanilla-closedcode na VS Code i popularne viljuške kao što su Cursor, Windsurf, VSCodium:

  1. Otvorite VS Code
  2. Otvorite integrirani terminal
  3. Pokrenite closedcode - ekstenzija se automatski instalira Ako s druge strane želite da koristite svoj vlastiti IDE kada pokrenete /editor ili /export iz TUI-ja, morat ćete postaviti export EDITOR="code --wait". Saznajte više.

Ručna instalacija

Potražite vanilla-closedcode na Extension Marketplaceu i kliknite na Instaliraj.

Rješavanje problema

Ako se ekstenzija ne uspije automatski instalirati:

  • Uvjerite se da koristite closedcode u integriranom terminalu.
  • Potvrdite da je CLI za vaš IDE instaliran:
    • Za VS kod: code naredbu
    • Za Cursor: cursor naredba
    • Za Windsurf: windsurf komanda
    • Za VSCodium: codium komanda
    • Ako ne, pokrenite Cmd+Shift+P (Mac) ili Ctrl+Shift+P (Windows/Linux) i potražite "Shell Command: Install 'code' command in PATH" (ili ekvivalent za vaš IDE)
  • Osigurajte da VS Code ima dozvolu za instaliranje ekstenzija
